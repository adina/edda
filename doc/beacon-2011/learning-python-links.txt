=========================================
Miscellaneous links on Python programming
=========================================

.. contents::

A complete list is `here
<http://wiki.python.org/moin/BeginnersGuide/Programmers>`__.  The
below list is slightly curated.

Introductory links
------------------

Practical Computing for Biologists (Haddock and Dunn_:

   http://practicalcomputing.org/

The Software Carpentry Introduction to Python:

   http://software-carpentry.org/4_0/python/intro/

Learning Python (Mark Lutz):

   http://oreilly.com/catalog/9781565924642

How to Think like a Computer Scientist (Allen B. Downey):

   http://greenteapress.com/thinkpython/thinkpython.html

Slightly more advanced
----------------------

Learn Python the Hard Way (Zed Shaw):

   http://learnpythonthehardway.org/

Dive Into Python (Mark Pilgrim):

   http://diveintopython.org/

   (This is recommended if you already know another programming language.)

Intermediate and Advanced Software Carpentry in Python (C. Titus Brown):

   http://ivory.idyll.org/articles/advanced-swc/ 

Great reference material
------------------------

The official Python documentation (for 2.x):

   http://docs.python.org/

   (especially the `tutorial
   <http://docs.python.org/tutorial/index.html>`__ and `standard
   library reference <http://docs.python.org/library/index.html>`__)
    

Python Cookbook:

   http://code.activestate.com/recipes/langs/python/
